Возвращает расширение файла, указанного через URL.

### Синтаксис
`UrlPathSuffix(url)`

### Аргументы
- **url** (тип: `string`)  
    URL, из которого извлекается расширение файла.

### Результат
- **String**  
    Расширение файла.

### Пример
```js
UrlPathSuffix('http://news.websoft.ru/db/kb/0939DD37D1C5F9B8C3257403003E8F4F/tree.html') // вернет '.html'
```
