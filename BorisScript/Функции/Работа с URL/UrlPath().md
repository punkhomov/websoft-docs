Извлекает путь из переданного URL.

### Синтаксис
`UrlPath(url)`

### Аргументы
- **url** (тип: `string`)  
    URL, из которого извлекается путь.

### Результат
- **String**  
    Путь из URL.

### Пример
``` js
UrlPath('http://news.websoft.ru/db/kb/0939DD37D1C5F9B8C3257403003E8F4F/tree.html') 
// вернет '/db/kb/0939DD37D1C5F9B8C3257403003E8F4F/tree.html'
```
