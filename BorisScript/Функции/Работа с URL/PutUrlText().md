Сохраняет содержимое строки в файл по заданному URL с использованием кодировки UTF-8 BOM.

## Синтаксис
`PutUrlText(url, text)` 

## Аргументы
- **url** (тип: `string`)  
    URL, по которому будет сохранён файл.
- **text** (тип: `string`)  
    Содержимое строки, которое будет сохранено.

## Результат
Возвращает `undefined`.
