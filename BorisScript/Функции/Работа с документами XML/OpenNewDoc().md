Создает новый XML-документ по заданной форме.

### Синтаксис
`OpenNewDoc( formUrl )`

### Аргументы
- **formUrl** (`String`): URL формы, по которой создается новый документ.

### Результат
- **XmlDoc**: Объект нового XML-документа.
