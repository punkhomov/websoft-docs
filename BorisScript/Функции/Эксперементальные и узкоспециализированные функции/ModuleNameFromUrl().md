Выдает наименование модуля, в котором находится файл, по заданному url со схемой x-app.

### Аргументы
- **url** (тип `String`)  
    URL файла

### Результат
- `string`

### Пример
```js
ModuleNameFromUrl('//rcr/rcr_old.js') // вернет 'rcr'
```
