Динамически регистрирует файл v2-библиотеки в приложении, как если бы он был описан в модуле через `<code_source_file>`.

### Синтаксис
`RegisterCodeLibrary(url)`

### Аргументы
- **url** (тип `String`)  
    URL файла библиотеки с расширением `.bs` или `.xmi`.

### Описание
Функция позволяет динамически добавить библиотеку в приложение. После регистрации, библиотека становится доступной для использования так же, как и библиотеки, указанные в модуле через `<code_source_file>`.

### Пример
```js
// Регистрация библиотеки по URL 
RegisterCodeLibrary('path/to/library.bs');
```
