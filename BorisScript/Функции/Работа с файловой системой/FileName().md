Извлекает имя файла из пути, переданного в качестве аргумента.

### Синтаксис
`FileName( url )`

### Аргументы
- **url** (String):  
    Путь к файлу.

### Результат
- **String**:  
    Возвращает имя файла или директории из указанного пути.

### Примеры
```js
FileName('work/data/doc.pdf') // 'doc.pdf' 
FileName('work/data/') // 'data'
```
