Удаляет определенные формы из кэша. Функция используется в редких случаях при изменении структур данных на лету.

### Аргументы
- **urlPattern** (тип: `string`)  
    Маска URL формы (т.е. XMD-файла).

### Результат
- `undefined`

### Пример
`DropFormsCache('*candidate*')`
