Регистрирует XML-форму, описанную в строке. Используется для программной генерации форм "на лету".

### Аргументы
- **formUrl** (тип `String`) 
	URL, по которому будет зарегистрирована форма
- **formData** (тип `String`) 
	Строка с описанием формы
