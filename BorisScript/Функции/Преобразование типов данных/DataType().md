Функция `DataType()` возвращает строку, содержащую тип данных переданного значения. Возможные значения:
- `"undefined"`
- `"null"`
- `"string"`
- `"integer"`
- `"real"`
- `"bool"`
- `"date"`
- `"object"`

## Синтаксис
``` DataType(value) ```

## Аргументы
- **value**
	Значение, тип которого необходимо определить.
