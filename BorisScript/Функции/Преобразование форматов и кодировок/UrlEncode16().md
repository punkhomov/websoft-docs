Кодирует строку символов для использования в качестве параметра URL, используя UTF-16.

### Аргументы
- **str** (тип: `String`)
	Строка для кодирования.

### Результат
- `String`

### Пример
```js
UrlEncode16('qwerty-апрол.htm') // 'qwerty%2D%u0430%u043F%u0440%u043E%u043B%2Ehtm'
```
