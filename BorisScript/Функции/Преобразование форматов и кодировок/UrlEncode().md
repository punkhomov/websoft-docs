Кодирует строку символов для использования в качестве параметра URL.

### Аргументы
- **str** (тип: `string`)
	Строка, подлежащая кодированию.

### Результат
- `string`
	Закодированная строка.

### Пример
```js
UrlEncode('qwerty-апрол.htm') вернет 'qwerty%2D%E0%EF%F0%EE%EB%2Ehtm'
```
